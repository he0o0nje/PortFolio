PROJECT 06

AuViS
(Auto Visual Scheduler)

Deep-Learning Team Project
(React-Native / Django / Tibero / VOLOv5, OCR / Figma)

Figma로 디자인 및 와이어프레임을 만든 후
React-Native로 Front-end를 구현하고,
Django로 Back-end를 구축하였습니다.

Tibero를 이용하여 데이터베이스를 만들었고,
Deep-Learning 모델 YOLOv5, OCR을 통해 추출한
데이터를 Triton server를 이용해서 연동하였습니다.

기본적인 캘린더 기능에 공고 포스터와 같이
일정이 나와있는 이미지를 업로드하면
자동으로 일정이 추가되는 어플입니다.
